# React Learning Projects

A collection of React projects built while learning React fundamentals and modern frontend development concepts. These projects cover state management, Context API, Redux Toolkit, React Router, hooks, and reusable components.

## Projects

### 🎨 bgChanger
Background color changer using React state.

### 🔢 counter
Simple counter application using React hooks.

### 💱 currencyConverter
Currency converter application with dynamic conversions.

### 🌐 miniContex
Practice project demonstrating React Context API.

### 🔐 passGenerator
Random password generator with customizable options.

### ✅ reduxToolkitTodo
To-Do application built using Redux Toolkit for state management.

### 🌙 themeSwitcher
Light/Dark mode theme switcher using React.

### 📝 todoContext
To-Do application managed with React Context API.

### 🛣️ webSiteWithRouter
Multi-page React website using React Router.

## Technologies Used

- React
- JavaScript (ES6+)
- React Hooks
- Context API
- Redux Toolkit
- React Router
- CSS

## Purpose

This repository contains projects created while learning React and exploring different approaches to state management, routing, and component-based development.
